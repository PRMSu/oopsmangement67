# 🧹 Cleaning Management System (OOP Project)

## 📌 Overview
This is a small Java-based project demonstrating Object-Oriented Programming concepts using a real-world cleaning management system.

## 🚀 Features
- Users can raise complaints
- Admin assigns workers
- Workers perform cleaning tasks

## 🧠 OOP Concepts Used
- Encapsulation (User class)
- Inheritance (Worker extends User)
- Polymorphism (Method overriding)
- Abstraction (Complaint class)

## 🛠️ Technologies
- Java

## ▶️ How to Run
1. Compile all files:
   javac *.java

2. Run:
   java Main
